## 📝 Про проєкт

### Яку задачу він вирішує?

Проєкт створений для презентації кондитерських виробів, замовлення десертів
онлайн та залучення нових клієнтів. Він допомагає користувачам швидко
ознайомитися з асортиментом, дізнатися ціни, склад солодощів та оформити заявку.

### Основний функціонал

- **Каталог товарів** — перегляд тортів та тістечок за категоріями.
- **Картка десерту** — детальний опис, інгредієнти та фотографії.
- **Форма замовлення** — швидка відправка заявки кондитерам.
